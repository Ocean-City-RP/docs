# Ocean City RP Core Framework

## Overview
Core framework providing essential functionality for FiveM servers.

### Key Features
- Player Management & Data Persistence
- Economy System
- Job System
- Status System (Health, Hunger, Thirst, Stamina)
- Admin Commands
- Database Integration
